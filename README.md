# Citizen Requests SPA

Одностраничное приложение (SPA) для визуализации обращений граждан: таблица + фильтры/поиск + карта + карточка обращения.

## Стек

- React + Vite (JavaScript)
- MUI
- React-Leaflet + Leaflet

## Запуск

```bash
npm install
npm run dev
```

## Данные

Данные из `public/data.json`.

## Возможности

- Фильтр по статусу
- Поиск по категории/адресу (с debounce)
- Сортировка колонок таблицы
- Пагинация (10 на страницу)
- Сводка (по статусам и топ категорий)
- Карта с маркерами + авто-приближение под текущую выборку
- Модальное окно с деталями
- Сохранение фильтров/сортировки/страницы в URL

## Артефакты сдачи

- **Таблица самооценки:** `docs/self-evaluation.md`
- **Скриншот результата MBTI:** `docs/mbti.png`
